Howdy, and welcome to my Unit Converters! 

This is a bundle of Matlab functions which all take the form from2to().   

EXAMPLE: Convert 700 kilopascals to pounds per square inch:

  kPa2psi(700)

  ans =

    101.5264 

To reduce clutter in your home folder, you may want to put all these functions in a folder called functions and create an m-file in your home folder called startup.m.  In startup.m write the line 

addpath('C:\myfilepath\functions')

and save it.  Each time you start Matlab, the program will then automatically read startup.m and be reminded to keep an eye on the functions folder.
